# 🌟 NextEraX - Blockchain-Enabled Tourism Platform

Platform : https://frontend-sepia-seven-9wck7zaulp.vercel.app/

Welcome to **NextEraX**! This is a revolutionary blockchain-based platform designed to transform the tourism experience by connecting tourists with verified local guides. Built with a stunning Sky Blue aesthetic and modern interactive UI, NextEraX provides immersive experiences and ensures secure transactions through smart contracts.

## ✨ What's New? (Recent Updates)
- **Sky Blue Aesthetic:** A completely overhauled, modern, and vibrant Sky Blue theme across all dashboards.
- **Premium Loading Animations:** A breathtaking orbital 3D loader featuring transparent animated bubbles and travel illustrations.
- **Interactive Guide Dashboard:** A brand new, fully interactive portal for guides to manage bookings, track analytics, and view their Soulbound NFT status.

## 🚀 Key Features

### 🛡️ For Tourists
- **Browse Verified Guides:** Ensure safety by booking guides verified on the blockchain.
- **Smart Contract Bookings:** Secure, automated escrow payments.
- **AR/VR & 360° Tours:** Immersive virtual exploration of destinations before visiting.
- **AI Assistant:** Get instant personalized travel itineraries.

### 🗺️ For Local Guides
- **Soulbound NFT Certificates:** Receive unforgeable, non-transferable digital credentials proving your expertise.
- **Manage Availability:** Accept or decline tour requests in real-time.
- **Real-time Analytics:** Track your total earnings in ETH, average ratings, and completed tours.
- **Secure Payouts:** Funds are released instantly via smart contracts after tour completion.

### ⚙️ For Administrators
- **Verify Applications:** Perform background checks and mint NFT certificates directly to guide wallets.
- **Monitor Analytics:** Visualize platform growth, total bookings, and revenue.
- **Resolve Disputes:** Manage a ticketing system for user reports.
- **Platform Control:** Adjust global parameters like Platform Fee Percentages and maximum daily bookings.

## 💻 Running the Interactive Demo

The platform currently includes robust **Interactive Mock Data** making it incredibly easy to demonstrate the UI without needing a live blockchain node.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/Jharkhand-Tourism.git
   cd Jharkhand-Tourism/frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the application:**
   ```bash
   npm start
   ```

4. **Testing the Dashboards:**
   - Click the user icon in the top-right navigation bar.
   - Use the **Demo Tourist**, **Demo Guide**, or **Demo Admin** buttons to instantly access the different portals.
